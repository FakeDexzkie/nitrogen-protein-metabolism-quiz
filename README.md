# Nitrogen/Protein Metabolism Quiz Website

This is a static browser-based multiple-choice quiz with 60 items.

## Files
- `index.html` — complete quiz website with questions, answers, score tracking, and explanations.

## How to deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then click **Save**.
6. Wait for GitHub Pages to publish the site.

The quiz shows one question at a time, checks the answer, displays the correct answer, and gives an explanation.
